This is a test application that allows someone to monitor multiple Mailinator accounts. The most likely use case scenario for this app is to check if your app is correctly sending out email. To use, enter the email addresses that you are sending to into the Mailinator Addresses box. Then have your app send to each of those email addresses (but strip off the normal domain and use mailinator instead. So jesse.vogt@gmail.com should become jesse.vogt@mailinator.com). You can set it up to filter on the email address you are sending to so you don't get too many other junk emails showing up. Let me know if you have any questions: jesse@shyknee.net.
